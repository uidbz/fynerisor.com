# fynerisor.com

Documentation website for Fynerisor - Risor scripting interface for Fyne GUI applications.

## Development

This site is built with Hugo and the Hextra theme.

### Prerequisites

- Hugo extended version
- Go 1.21+

### Running locally

```bash
hugo server
```

Visit http://localhost:1313

### Building

```bash
hugo
```

Output will be in the `public/` directory.

## Content

The site includes:
- Getting started guide
- API reference for widgets, containers, and global objects  
- Examples and tutorials
- Advanced topics (data binding, authentication, etc.)
